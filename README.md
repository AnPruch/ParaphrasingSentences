# Проект ПДЛ: Перефразирование предложений

В рамках проекта планируется настройка автоматического перефразирования предложений через изменение их синтаксической и лексической наполненности. 

На вход подаётся одно предложение, на выход — несколько вариантов перефраза различной структуры и семантики.
