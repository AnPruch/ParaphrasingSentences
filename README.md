Проект ПДЛ: Перефразирование предложений
========================================

В рамках проекта осуществлена настройка автоматического перефразирования предложений через изменение их синтаксической и лексической наполненности. 

На вход подаётся одно предложение, на выход — несколько вариантов перефраза различной структуры.

Данный проект может быть использован для удобства перифраза, например, промптов для передачи языковым моделям.

Структура
---------

Проект состоит из трёх блоков:

 - Модель для перифраза  (на основе трансформера
 ``Рegasus <https://huggingface.co/docs/transformers/main/en/model_doc/pegasus#transformers.PegasusConfig>``__) в файле ``src/paraphraser.py``;
 
 - Функции для работы с датасетом: save2json и json_load в файле ``src/dataset_manager.py``;

 - Пример работы проекта в файле ``src/start.py``.  (Пример работы лучше хранить в папке `tests` и с соответствующим названием - `usage` или `example`)

Запустить пример работы можно через команду ``python ./src/start.py``. (Если это все-таки не пример, а точка входа, то так и пишите)

Тесты
-----

В проекте присутствуют тесты на корректную работу модели перефразирования.

Их можно запустить через команду ``pytest .\tests``.
